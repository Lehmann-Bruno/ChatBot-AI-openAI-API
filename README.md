# ChatBot-AI-openAI-API
AI chatbot for college project, using JS and openAI api.

.env values:
OPENAI_API_KEY=
BUSINESS_NAME=

before running it you need to do "npm install" at your project folder in cmd,
after that just run "node index.cjs" for it to work

IMPORTANT: the first time you run you need to connect it to an whatsapp account via QRcode, you won't lose any data but all messages you receive will trigger the bot.
